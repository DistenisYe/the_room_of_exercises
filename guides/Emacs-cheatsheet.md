# Emacs 快捷键

Mac下面大多数地方都可以用这些快捷键


### SHORTCUTS

| Key/Command | Description |
|---|---|
| Ctrl + f | 光标向前(Forward)移动一个字符位置 |
| Ctrl + b | 光标往回(Backward)移动一个字符位置 |
| Ctrl + p | 相当于通常的上箭头 |
| Ctrl + n | 相当于通常的下箭头 |
| Ctrl + d | 删除一个字符，相当于通常的Delete键 |
| Ctrl + h | 退格删除一个字符，相当于通常的Backspace键 |
| Ctrl + u | 删除光标之前到行首的字符 |
| Ctrl + k | 删除光标之前到行尾的字符 |
| Ctrl + a | 光标移动到行首，相当于通常的Home键 |
| Ctrl + e | 光标移动到行尾 |
| Ctrl + r | 可以搜索历史命令 |
| Ctrl + l | 类似 clear 命令效果 |
| Ctrl + y | 粘贴 |
| Ctrl + t | 交换光标处文本 |
