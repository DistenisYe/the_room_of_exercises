## JavaScript 对象类型的确定

typeof 可以确定对象的基本类型: number, string, object, function, undefined。 === 比较值和对象类型是否相等。 == 仅比较值是否相等。

=== 比较的是基本类型，如果是对象的子类型，同样满足。所以 1 === 1.0 是 true 而 1 === "1" 是false。

## if 语句

}
